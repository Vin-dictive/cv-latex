# LaTeX CV Template

A minimal, professional CV template in LaTeX.

## Usage

1. Edit `cv.tex` with your information
2. Compile with: `make` or `pdflatex cv.tex`
3. Clean build files: `make clean`

## Requirements

- LaTeX distribution (MacTeX, TeX Live, or MiKTeX)
- pdflatex compiler

## Customization

- Modify sections in `cv.tex` as needed
- Adjust margins and spacing in the preamble
- Add or remove sections based on your needs
